About

Cool Kiwi Kids is an online shop that imports and sells pretty cool 3d shaped kids helmets from Denmark. I was approached to create an E-commerce website for them. Cool Kiwi Kids needed an online platform to sell the helmet and additional accessoires like drinking bottles, gloves, locks, saddle covers and their latest addition backpacks. After research as to what tool to use I suggest either Wix or Shopify. Wix is an Israeli software company, providing cloud-based web development services. It allows users to create HTML5 websites and mobile sites through the use of online drag and drop tools. Shopify is a Canadian multinational e-commerce company headquartered in Ottawa, Ontario. It is also the name of its proprietary e-commerce platform for online stores and retail point-of-sale systems. As Wix has a lower margin on sold products the customer agreed to set up a Wix site.

The Idea

The idea was to create a web site kids and parents will find appealing and present Cool Kiwi Kids products in a fashionable manner not only in the looking but also in the functionality.

Technical specifications

As Wix is basically a drag and drop application and you get what you see there are not many technical specification. The most challenging part was probably tho get the SEO right but Wix provides plenty of tools for that as well. Another interesting aspect was to create the online shop and again Wix makes this a breeze. In a recent release one can add JavaScript to the site to make it more interactive and approachable for the customer. Also the new Code Editor X comes in very handy. Please have a look here and let me know what you think.

Wire frames & Navigation

The functionality of the web page was quite simple really. From the landing page all the links where clear and bold and would help navigate trough out the entire site. Images acted as an introduction for the content and guide the user through the website.
