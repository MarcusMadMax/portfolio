About

Share Me was my submission for the "App Development" summative. The wire framing and design was done in Adobe XD. I used HTML plus SCSS for the frame wrok and JavaScript to build UI components and enhance user experience like a modern web app.

The Idea

The idea was to create a sharing app similar to Facebook© or Instagramm© where the user can upload images and make comments as well as recieve feedback from other participants.

Technical specificatios

As this was a team project we had to collaborate as well as contribute actively as individuals and function as a collective to make the project a success. We decided to use React in favor to other JavaScript frameworks because it is part of the MERN Stack we planed to integrate in our project and it consists of Mongo DB, Express.js, React and Node.js. Why we used the MERN Stack you ask? Simply because it’s main programming language is JavaScript and we know how to write JavaScript, well sort of cause we are still learning.

Wireframes & Navigation

The functionality of the app was straight forward. A Log in and sign up screen welcome the user and enable a more personalised user expirience. Navigational elements can be found easily and bound to be self explanatory.
