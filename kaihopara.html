About

Kaihōpara NZ was my submission for the "Foundation Coding" summative. The wire framing and design was done in Adobe XD. I used HTML plus SCSS for the frame work and JavaScript to build UI components and enhance user experience like a modern web app.

The Idea

The idea was to create a play full easy to use and visually appealing web app that can be used by locals and tourists alike.

Ethics

Part of the brief was to implement Maori culture heritage and lifestyle. Hence the name Kaihōpara which translates to explorer. Also the choice of colour supposed to reflect the connection between Papatūānuku, earth mother, and Ranginui ,sky father, from Maori mythology.

Wire frames & Navigation

The functionality of the app was straight forward. A Log in and sign up screen welcome the user and enable a more personalized user experienced. Navigational elements can be found easily and bound to be self explanatory.
